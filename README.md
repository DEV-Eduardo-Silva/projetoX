# Sistema de Gestão de Oficina

Sistema desenvolvido em Python com Streamlit para controle de ordens de serviço, cadastro de placas e acompanhamento operacional.

---

# Objetivo

Facilitar o gerenciamento da operação da oficina, permitindo controle em tempo real das ordens de serviço, equipes e indicadores.

---

#  Funcionalidades

* 📋 Cadastro de placas
* 🧑‍🔧 Abertura de ordens de serviço
* ⏱️ Controle de início e fim dos serviços
* 📊 Dashboard com indicadores
* 📍 Controle de status (aberto, em andamento, finalizado)

---

# Tecnologias utilizadas

* Python
* Streamlit
* PostgreSQL
* psycopg2
* python-dotenv
